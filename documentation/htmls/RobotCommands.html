Robot Komutları
Aşağıdaki resimde robotumuz üzerinde yer alan sensörler ve bu sensörlerin komutları yer almaktadır. Robotumuzun üzerinde 4 adet mesafe sensörü ve ve alt köşelerinde yer alan siyah ve beyaz rengi ayırt edebilen 4 adet renk sensörü bulunmaktadır.

Şimdi robotun komutlarını açıklamalarıyla birlikte inceleyelim:

Python Kodu Açıklaması
robot.is_ok() Robotun ve simülasyonun çalışması durumunda True, çalışmıyor olması durumunda False değerini alır.
robot.move(x) Robotu ileriye doğru x ile girilen nümerik değer hızında (m/s) hareket ettirir. x pozitif ise robot ileriye gider, negatifse geriye doğru gider.
robot.rotate(x) Robotu olduğu yerde x ile girilen nümerik değer hızında (radyan/s) döndürür. x negatif ise robot saat yönünde, pozitifse ters yönde döner.
robot.sleep(x) x saniye süre boyunca kodu uyutur.
robot.time() Roundda o ana kadar geçen süreyi veren komuttur.
robot.round() Müsabakanın kaçıncı roundda olduğunu veren komuttur.
robot.front() Ön mesafe sensörünün verisini metre cinsinden float değer olarak verir.
robot.left() Sol mesafe sensörünün verisini metre cinsinden float değer olarak verir.
robot.right() Sağ mesafe sensörünün verisini metre cinsinden float değer olarak verir.
robot.back() Arka mesafe sensörünün verisini metre cinsinden float değer olarak verir.
robot.front_left_color() Robotun sol ön renk sensörünün algıladığı renk bilgisini 0 (siyah) ve 1 (beyaz) olarak verir.
robot.front_right_color() Robotun sağ ön renk sensörünün algıladığı renk bilgisini 0 (siyah) ve 1 (beyaz) olarak verir.
robot.back_left_color() Robotun sol arka renk sensörünün algıladığı renk bilgisini 0 (siyah) ve 1 (beyaz) olarak verir.
robot.back_right_color() Robotun sağ arka renk sensörünün algıladığı renk bilgisini 0 (siyah) ve 1 (beyaz) olarak verir.
robot.body_width(x) Robotun genişliğini x metre yapan komuttur. x değeri 0.05 - 0.1 metre aralığında olabilir.
robot.body_height(x) Robotun yüksekliğini x metre ölçüye ayarlayan komuttur. x değeri 0.05 - 0.1 metre aralığında olabilir.
robot.body_length(x) Robotun uzunluğunu x metre ölçüye ayarlayan komuttur. x değeri 0.05 - 0.2 metre aralığında olabilir.
robot.wheel_radius(x) Robotun tekerlerinin çapını x metre ölçüye ayarlayan komuttur. x değeri 0.01 - 0.04 metre aralığında olabilir.
robot.wheel_thickness(x) Robotun teker kalınlığını x metre ölçüye ayarlayan komuttur. x değeri 0.01 - 0.04 metre aralığında olabilir.
robot.bumper_radius(x) Robotun tampon çapını x metre ölçüye ayarlayan komuttur. x değeri 0.015 - 0.05 metre aralığında olabilir.
robot.bumper_length(x) Robotun tampon uzunluğunu x metre ölçüye ayarlayan komuttur. x değeri 0.02 - 0.18 metre aralığında olabilir.
